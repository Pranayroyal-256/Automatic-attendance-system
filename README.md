\# Automatic Attendance System Using Face Recognition



A face recognition based attendance monitoring system using OpenCV and KNN algorithm.



\## Features

\- Face detection using Haar Cascade

\- Face recognition using KNN classifier

\- Automatic attendance marking

\- CSV attendance records

\- Streamlit dashboard support

\- MySQL database integration



\## Technologies Used

\- Python

\- OpenCV

\- Scikit-learn

\- Streamlit

\- MySQL
